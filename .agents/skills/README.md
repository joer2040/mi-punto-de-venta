# Local Skills

This repo has local skills in `.agents/skills`.

Available skills:
- `systematic-debugging`
- `commit-work`
- `session-handoff`
- `agent-md-refactor`
- `naming-analyzer`

Use them when the user asks for the skill by name or when the task clearly matches the skill purpose.

Project context:
- Frontend: React + Vite
- Main validation command: `npm.cmd run build`
- Backend: Supabase Edge Functions and SQL migrations
- Sensitive areas: `src/pages`, `src/api`, `supabase/functions`, `supabase/migrations`
